Retro Georgy Alarm Clock Web Installer

WLED-style browser flashing for ESP32-S3 (MicroPython firmware).

  1. Connect your ESP32-S3 by USB.
  2. Open this page in Chrome or Edge over HTTPS.
  3. Click install and select your serial port.
  4. After flashing, deploy project files with mpremote or OTA.

Configured firmware: ESP32_GENERIC_S3-SPIRAM_OCT-20260406-v1.28.0.bin

This browser does not support Web Serial. Use Chrome or Edge. Open this page over HTTPS (or localhost) to use Web Serial.

Firmware manifest: web-installer/manifest.json
